IMAGE SETUP INSTRUCTIONS
========================

To add your photos to the gallery:

1. Save your 3 provided images as:
   - images/photo1.jpg (friends studying together)
   - images/photo2.jpg (friends having meal) 
   - images/photo3.jpg (friends playing Mahjong)

2. To add 3-4 more images, replace the placeholder divs in index.html:
   - Find the placeholder divs (lines ~42-65)
   - Replace each placeholder with a regular photo-item div like:
   
   <div class="photo-item">
       <img src="images/photo4.jpg" alt="Your description" class="gallery-img">
       <div class="photo-overlay"></div>
   </div>

3. The theme images (Hello Kitty, Bubble Tea, Cupcakes, Beijing) use online sources.
   If they don't load, you can replace the img src URLs with local images in the images/ folder.
